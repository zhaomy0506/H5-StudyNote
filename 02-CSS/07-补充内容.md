# 2.css变量

css中如果有一些样式值,需要频繁使用,那么可以交给变量储存

## 2.1声明css变量

CSS变量的声明需以`--`为前缀，并使用`var()`函数进行引用。

```css
:root {
    --font-color: #ff6700;
}

p {
    color: var(--font-color,#fff);/*逗号后,传入默认值*/
}
```

作用范围: 声明的变量仅在其所在的容器内部起作用。

若需声明全局变量，应将变量声明在根容器 `:root` 中。
