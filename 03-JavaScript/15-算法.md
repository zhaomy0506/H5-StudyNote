## 冒泡排序和比较排序区别

二者区别在于

- 一个是与自身相邻的比较
- 一个是拿自身向后比较

```js
//冒泡排序
const arr = [3, 2, 4, 1, 5, 7, 9, 6, 0, 8]
//控制比较的轮数
for (let i = 0; i < arr.length; i++) {
    //拿当前索引和索引+1比较,索引一直在变(j++)
    for (let j = 0; j < arr.length - 1; j++) {
        if (arr[j] > arr[j + 1]) {
            let flag = arr[i];
            arr[j] = arr[j + 1];
            arr[j] = flag;
        }
    }
}
```

```js
//选择排序
const arr = [3, 2, 4, 1, 5, 7, 9, 6, 0, 8]
//控制用谁(arr[i])比较,每轮i+1
for (let i = 0; i < arr.length; i++) {
    //控制跟谁(arrj)比较
    for (j = i + 1; j < arr.length; j++) {
        if (arr[i] > arr[j]) {
            let flag = arr[i]
            arr[i] = arr[j]
            arr[j] = flag
        }
    }
}
```
